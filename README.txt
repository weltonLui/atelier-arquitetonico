ATELIER ARQUITETÔNICO — Mini site (1 página)

Como usar (bem simples):
1) Abra 'index.html' no navegador (clique duplo).
2) Para publicar:
   - Suba a pasta inteira para um serviço de hospedagem estática (Netlify, Vercel, GitHub Pages).
   - Ou mande para quem faz seu site.
3) Links importantes ficam dentro de index.html (WhatsApp/Instagram/e-mail).

Observação:
- O botão "Baixar Portfólio PDF" aponta para o PDF dentro de /assets.
